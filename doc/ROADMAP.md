# **Project SNN4: 統合版戦略的ロードマップ (v2.0)**

## **エグゼクティブサマリー：新学習則の解明と創発的知能の実現**

本ロードマップは、誤差逆伝播法（BP）に頼らない**新学習アルゴリズムの理論的基礎固め**を短期目標とし、その成果を基盤としてSNN固有の優位性（超低消費電力、継続的適応、因果的推論）を確立する\*\*創発的知能（Emergent Intelligence）\*\*の実現を長期目標とします。

## **Phase 1: コア学習メカニズムの理論的深化と検証（1〜3ヶ月）**

**目的**: 新アルゴリズムの仕組みを理論的に解明し、再現性と頑健性を保証する。

| ID | 課題カテゴリ | 具体的タスク | 補足と意図 |
| :---- | :---- | :---- | :---- |
| **P1-1** | **理論的裏付け** | **予測符号化（PC）との理論的接続の明確化** | 新アルゴリズムの更新規則が、PCやTarget Propagationといった非BP型の信用割当メカニズムの枠内でどのように位置づけられるかを数学的に証明する。 |
| **P1-2** | **メカニズム検証** | **活性化関数の位置逆転の影響調査** | 「入力 $\\rightarrow$ 活性化 $\\rightarrow$ 積和演算」という位置逆転が、誤差信号の伝播と学習に与える効果を定量的・定性的に検証する。 |
| **P1-3** | **安定性の確立** | **ハイパーパラメータ感度の検証** | 学習率やバッチサイズ、重み初期値などに対する新アルゴリズムの感度を広範囲にテストし、頑健で汎用性の高い推奨設定を確立する。 |
| **P1-4** | **コード抽象化** | **新学習アルゴリズムのモジュール化と抽象化** | 現在のアルゴリズム実装を、他のNNアーキテクチャ（CNN、RNNなど）へ容易に応用できるよう、コアロジックを独立した抽象クラスとして再設計する。 |

## **Phase 2: ハードウェア適合性の改善と応用展開（4〜6ヶ月）**

**目的**: FPGA上の精度低下を克服し、LLM開発に不可欠な応用モデル（RNN、Attention）への展開を完了する。

| ID | 課題カテゴリ | 具体的タスク | 補足と意図 |
| :---- | :---- | :---- | :---- |
| **P2-1** | **FPGA精度改善** | **ハードウェアを意識した学習（Hardware-aware training）の導入** | FPGAでの精度低下（20〜30%）を解決するため、学習段階で量子化やLUTの制約をシミュレートする機構を導入し、GPU学習と実機精度の乖離を最小化する。 |
| **P2-2** | **RNN・時系列応用** | **新アルゴリズムによるRNNの実装と検証** | LLM開発の基盤となるRNN構造（時系列データ処理）に新アルゴリズムを適用し、BPTTベースのRNNと同等以上の性能と計算効率を実証する。 |
| **P2-3** | **LLM要素開発** | **CBOWおよびAttention機構への適用** | 言語モデルのコア要素であるAttention機構の学習に新アルゴリズムを適用し、基礎的な自然言語処理タスクでの効率性を検証する。 |
| **P2-4** | **ベンチマーク強化** | **ANN vs SNN リーダーボードの自動化** | 精度、速度、エネルギー効率の3軸での比較結果を自動で集計・追記するベンチマークスイートを完成させ、プロジェクトの優位性を継続的に発信する。 |

## **Phase 3: 継続学習と自律的進化機構の統合（7〜12ヶ月）**

**目的**: 新アルゴリズムを継続学習・進化の枠組みに統合し、AGIの中核となる自己修正能力を獲得する。

| ID | 課題カテゴリ | 具体的タスク | 補足と意図 |
| :---- | :---- | :---- | :---- |
| **P3-1** | **進化戦略の統合** | **ES（進化的アルゴリズム）を用いたハイブリッドNASの構築** | ESをメタ学習レイヤーとして導入し、ESがモデルの「アーキテクチャ」（層数、次元、接続性）を、新アルゴリズムが「重み」を最適化する統合フレームワークを完成させる。 |
| **P3-2** | **継続学習（CL）** | **新アルゴリズムによるSNNの**忘却しない**継続学習（CL）実装** | **Catastrophic Forgetting**（破滅的忘却）を回避するため、新アルゴリズムの局所的学習特性を利用した、新しい**記憶保持メカニズム**を設計・実装する（旧ロードマップF3-1を強化）。 |
| **P3-3** | **メタ認知実装** | **自己評価関数（情動）によるアーキテクチャ修正の実装** | エージェントの性能（例：予測誤差、環境報酬）を評価する「情動」モジュールを設計し、その評価結果に基づいてESがアーキテクチャ変更を行うロジックを実装する。 |
| **P3-4** | **真のLLM着手** | **SNNベースのTransformer/Mambaによる小規模言語モデルの訓練開始** | 構築した学習基盤と進化機構を用いて、小規模ながらも完全なSNNベースのTransformerまたはMambaモデルを訓練し、その効率性（パラメータ数あたりの性能）を検証する。 |

## **Phase 4: 因果的推論と高度な社会性の獲得（13〜18ヶ月）**

**目的**: AIに論理性、説明可能性、社会性を持たせ、創発的知能を現実のものとする。

| ID | 課題カテゴリ | 具体的タスク | 補足と意図 |
| :---- | :---- | :---- | :---- |
| **P4-1** | **因果的推論の強化** | **SNNによる因果的検索（Causal Search）アルゴリズムの実装** | 関連性（相関）だけでなく、原因と結果（因果）を区別するためのSNN独自の推論モジュールを開発し、計画や意思決定の精度を飛躍的に向上させる。 |
| **P4-2** | **協調戦略** | **マルチエージェント協調戦略と通信プロトコルの標準化** | 複数のDigitalLifeForm間のタスク**分割**、**交渉**によるリソース割り当て、**役割分担**といった高度な協調戦略を実装する（旧ロードマップF4-3を統合）。 |
| **P4-3** | **脳型XAI** | **内部的な因果連鎖に基づく**判断根拠の言語化**プロトタイプ構築** | 実行された行動に対し、内部的な因果連鎖（スパイクの連鎖）を基に、AIがその判断根拠を言語で説明するプロトタイプを構築する（旧ロードマップF4-5を統合）。 |
| **P4-4** | **公開と特許** | **技術公開に向けた理論論文と特許戦略の最終決定** | 新アルゴリズムの理論的優位性を示す論文を執筆するとともに、特許戦略（国際出願、企業との連携）を最終決定し、技術の適切な公開プロセスへ移行する。 |

## **Phase 5: ニューロモーフィック統合と実社会応用（18ヶ月〜）**

**目的**: SNNの究極的なポテンシャルを解放し、実機への展開と自律的な社会性を実現する。

| ID | 課題カテゴリ | 具体的タスク | 補足と意図 |
| :---- | :---- | :---- | :---- |
| **P5-1** | **実機最適化** | **Loihi/Akida向け最適化とEnergy-Efficiency-Ratio（EER）の計測** | モデルをニューロモーフィックハードウェア固有のフォーマットに変換し、実機で Energy per Inference を計測。**エネルギー効率と精度の積 (EER)** を最大化するモデルを探索する（旧ロードマップF5-1を統合）。 |
| **P5-2** | **社会学習** | **自律的な社会学習エコシステムの完成** | DigitalLifeFormが内発的動機に基づき、Distributed Model Registryを探索して他エージェントが公開した有用な知識・スキルを取り込む**社会性**を獲得する。 |
| **P5-3** | **イベントベースビジョン** | **DVSカメラなどイベントベースデータへの本格適用** | 高速かつ低遅延な処理が可能なイベントベースビジョンデータセットでのSNN4-HybridNetの性能を最大化し、リアルタイムロボティクスへの適用性を検証する。 |
